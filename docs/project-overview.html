Features

What it does

Core capabilities shipped or in active development.

Real-time streaming
Chunk-by-chunk AI responses via Server-Sent Events with abort support per request.
Conversation management
Create, list, rename, and resume conversations with LLM-generated auto-titles.
Multi-model selection
Switch between Google Gemini and Claude models live from a dropdown in the chat UI.
Secure authentication
JWT tokens in httpOnly cookies. No tokens in localStorage. CSRF-safe by design.
User preferences
Custom system instructions, accent color picker, font size — all persisted locally.
API key vault
Bring-your-own keys for LLM providers, stored encrypted at rest with Fernet.
Electron desktop shell
Same Next.js frontend wrapped for desktop. IPC-bridged capabilities with web fallbacks.
Whimsy generator
Collision-safe procedural doodle placer for expressive, branded empty states.
Knowledge & Channels
Structured knowledge bases and channel-based conversations beyond DM-style chat.
Projects
Group conversations and knowledge under projects for organised workspaces.

Quality & CI

How quality is enforced

Gates run on every PR to main. Self-hosted runner pool on VPS.

Biome
Lint + format for TypeScript/TSX. Zero lint suppressions without justification.
ruff + mypy
Python lint, format, and strict type checking via just check-all.
Sentrux
Architectural layer enforcement. Violations block PR merge. Baseline 6753/10000.
Stagehand E2E
LLM-driven browser tests via Stagehand v3 + GPT-4.1-mini. just stagehand-e2e.
File-line budget
500-line hard ceiling per .ts/.tsx/.py file. CI fails on overflow.
Design lint
bun run design:lint validates DESIGN.md token spec in CI on every PR.
